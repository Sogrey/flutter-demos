## Getting Started

For help getting started with Flutter, view our online [documentation](https://flutter.io/).

命令行flutter 安装运行

> flutter run
>
> flutter run --release

如果有多台设备连接，使用：

> flutter run -d \<deviceId\>

命令行flutter 打包apk

> flutter build apk
>
> flutter build ios //需要MAC支持

终止批处理操作：

> ctrl + c
>
> 终止批处理操作吗(Y/N)? y
